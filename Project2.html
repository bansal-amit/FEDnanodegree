Lessson 1 : The Basics of Web and HTML

How the Web Works

The web is a collection of computers or terminals that communicate with each other. There are four primary components involved in the web ecosystem : the user, browser, http and the server. The user uses the browser to send requests via http to the server. The http is the web's protocol that is used for communication between the browser and server.

What is HTML

HTML is acronym for Hyper Text Markup Language. HTML is the language for writing any web page. HTML5 is the latest version.The web pages are written using HTML tags, attributes and elements

What are HTML Tags and Elements

HTML documents are made of HTML elements. An HTML element is comprised of a starting tag, content and a closing tag. When writing HTML, we tell browsers the type of each element by using HTML tags. This video explains more about this. Broadly, there are either inline or block tags.

Inline vs Block tags

Examples of inline tags are b, strong, em, span. Examples of block tags are p, form and div. More information about the above tags can found here

Summary

In essence, we can summarise HTML, CSS and DOM as:

  1. HTML controls the "structure" of a web page
  2. CSS controls the "style" of a web page viz., how the page looks
  3. DOM is the reference to the tree-like structure of the web page

Design to First Draft

When you get the design from your company's graphic designer in the form of image/pdf file(s), the goal is to convert it to a web site. This site would contain all the necessary components i.e., .html, .css and .js files

Think of a web site like your house. The html files are the structure that define where the walls are placed, what room is the living room, what room is the kitchen, etc... Similarly, css is the style of the house i.e., color and/or decorations (wall color, wall hangings or other decorations). Finally, javascript files are the interactive components (something like a garage door opener or a TV remote) - these components change some element of the house.

More about HTML

For understanding and coding purposes, HTML should be thought of as a language. The basic word in html language is a tag. The browser turns the tag into elements that combine to form a tree. An element has a start tag, end tag and may or may not have content (i.e., text) and/or could even nest another element. An element has a name, type that determines what type of element will be created. When starting out from transitioning graphic designs into websites, two basic steps to keep in mind are

  1. Boxify the design
  2. Create a div,content and /div element for each box

CSS Vocabulary

CSS stands for Cascading Sytle Sheets. The word 'Style' can refer to many things and so can be confusing. It could mean HTML element or an attribute or even the general look of a web page. Rule is a line of CSS code describing the value that a certain attribute should take. Selector is the name that one uses in order to target the elements that are assigned to as a class or id attribute in HTML.

Box Model

Content is image or text that appears on the web page. think of this as the nucleus. This sits at the very center and the other layers serve to protect it so the other items don’t overlap with this. Padding clears an area around the content. this is affected by the background color of the box. Think of it as the space inside of the box that protects the content directly.Border goes around the padding and the content - this is inherited from the color property of the box. Lastly, <margin is the outermost layer and clears an area around the border. The margin does not have a background color - its completely transparent; this can be thought of as the space between the boxes.
